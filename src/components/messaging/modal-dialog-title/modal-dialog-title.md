# `modal-dialog-title`

Display a title within a modal dialog in a standardised format.

## Slots

### `default`

The content of the dialog title.

## Props

### `tag`

- type: `string`
- default: `h2`

The tag to use for the dialog title.

## Examples

```html
<modal-dialog-title>
	Delete user {{ username }}
</modal-dialog-title>
```
