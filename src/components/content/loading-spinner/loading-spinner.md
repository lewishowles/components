# `loading-spinner`

`loading-spinner` provides a standardised, spinning loading icon, primarily used in `loading-indicator`.

## Styling hooks

| Attribute                          | Element | Notes                          |
| ---------------------------------- | ------- | ------------------------------ |
| `data-component="loading-spinner"` | Root    | Scope styles to this component |

## Examples

### As part of an indicator

```html
<loading-spinner />
Loading data…
```
